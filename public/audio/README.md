# Audio Assets

Place your audio files in this directory:

## Optional File

- **birthday.mp3** - Background music
  - Format: MP3
  - Recommended: Instrumental, upbeat but not overwhelming
  - Volume: Will be set to 30% by default
  - Loop: Will loop continuously when playing

## Behavior

- Music is **OFF by default** (no autoplay)
- User can toggle music on/off via navbar button
- If `birthday.mp3` is missing, the music toggle button will be hidden
- The website works perfectly fine without background music

## Suggestions

Free music sources:
- [Pixabay Music](https://pixabay.com/music/)
- [Free Music Archive](https://freemusicarchive.org/)
- [YouTube Audio Library](https://www.youtube.com/audiolibrary)

Make sure you have the rights to use any music you add!
