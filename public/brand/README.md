# Brand Assets

Place your brand assets in this directory:

## Required Files

- **logo.png** - Company logo (transparent PNG recommended)
  - Recommended size: 200x200px or larger
  - Format: PNG with transparency
  - Used in: Navbar (40x40px display)

- **mr-martin.jpg** - Photo of Mr. Martin Marta
  - Recommended size: 800x800px or larger
  - Format: JPG or PNG
  - Aspect ratio: Square (1:1) for best results
  - Used in: Spotlight section

## Graceful Fallbacks

If these files are missing, the website will:
- **logo.png**: Hide the logo gracefully (no broken image)
- **mr-martin.jpg**: Show a placeholder with initials "MM"

The website will not crash if assets are missing.
